# sql-security-filtering-analysis

## Objetivo

Aplicar filtros en consultas SQL para investigar posibles incidentes de seguridad y obtener información específica sobre empleados dentro de un entorno organizacional simulado.
El objetivo fue utilizar cláusulas de filtrado para recuperar únicamente los datos relevantes según condiciones de tiempo, ubicación, departamento y estado de inicio de sesión.

### Habilidades Desarrolladas
- Construcción de consultas SQL con SELECT y FROM
- Uso de cláusulas WHERE para filtrado de datos
- Aplicación de operadores lógicos (AND, OR, NOT)
- Uso del operador LIKE y comodines (%)
- Análisis de intentos de inicio de sesión sospechosos
- Filtrado de datos orientado a investigación de seguridad

### Herramientas utilizadas
- Entorno de laboratorio SQL (simulado)
- Operadores SQL
